# SiliconSavant

## About

**SiliconSavant** is an educational repository dedicated to semiconductor and VLSI engineering. It provides practical resources that help students, researchers, and electronics enthusiasts understand device physics, circuit simulation, and semiconductor manufacturing through hands-on examples.

* Electronics & Communication Engineering (ECE) students
* Electrical Engineering students
* VLSI enthusiasts
* Embedded Systems learners
* Researchers and hobbyists

## Goals

* Make semiconductor concepts easier to understand.
* Provide practical LTspice simulation examples.
* Share MOSFET models and circuit analysis.
* Explain semiconductor fabrication with clear process flows.
* Build an open learning resource for the electronics community.
